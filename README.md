# muos-blue-archive-arona
Blue Archive - Arona theme for MuOS

It is better to enable animated background in settings so Arona can change her position.

## Supported resolutions:
- 640x480
- 720x480
- 720x720

## Tested devices
- Anbernic RG35XX plus
